# 구현할 기능 목록

## 사용자의 행동

1. 메인페이지에 들어온다.

- 상품 목록을 조회한다.

2. 장바구니 추가 버튼을 누른다.

- 장바구니에 새로운 상품이 추가된다.
- 상품 개수 버튼으로 바뀐다.
- 헤더의 장바구니에 담긴 상품 개수가 바뀐다.

3. 추가한 상품의 개수를 늘린다.

- 장바구니에 상품 개수가 늘어난다. (보이지는 않음)

4. 추가한 상품의 개수를 0으로 줄인다.

- 장바구니에 상품이 제거된다.
- 장바구니 추가 버튼으로 바뀐다.
- 헤더의 장바구니 상품 개수가 바뀐다.

5. 장바구니 목록에 들어온다.

- 장바구니에 담긴 상품 목록 및 수량을 조회한다.
- 결제 정보(총 상품가격 + 총 배송비 = 총 주문금액)를 조회한다.
  - 배송비는 3만원 이상 주문하면 0원이라고 가정한다.

6. 장바구니 목록에서 특정 상품의 수량을 변경한다.

7. 장바구니 목록에서 특정 상품을 제거한다.

8. 장바구니 목록 페이지에서 주문한다.

- 체크박스를 눌러 장바구니의 각 상품을 선택한다.
- 주문하기 버튼을 눌러 주문한다.
- 주문이 끝나면 주문 결과를 보여주는 페이지에 들어온다.
